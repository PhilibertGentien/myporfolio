Concept du site

Hello voici le site qui récapitule tout ce que j'ai pu faire en HTML CSS PHP JS.

Les projet sont toujours accompagnés de leur code source. Si tu trouve un problème ou une amélioration que je peux faire n'hésite pas a me le faire savoir soit en allant sur mon discord soit en remplissant le formulaire de contact.

Il se peut que j'ai laissé des fautes de Français dans ce que tu vois sur le site donc si tu en vois une dis le moi.

Tu peux aussi trouver des erreurs dans le code que je te met à disposition sache qu je ne suis qu'un humain et donc que cela peut arriver.
